Class DataJoint<TIn, TOut>

Data joint connects data sink with data faucet.

The data sank to the sink is poured to the faucet.

Type Parameters

  • in TIn extends TOut

    Type of data values accepted by joint sink.

  • out TOut = TIn

    Type of data values poured by joint faucet.
